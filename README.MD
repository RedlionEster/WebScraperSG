# 🎾 Historical Data Fun Project

Welcome to our awesome project! Here, we dive into the world of ATP and WTA tennis, mixed with some cool GDP and population stats. It's all about finding the hidden stories in the numbers and sharing some neat insights.

## What's Inside? 🧐

- **Epic Data Analysis**: Data analysis and comparison and a decade-long look at ATP and WTP.
- **Data Treasure Hunt**: Scraping historical data for WTA, ATP, GDP, and population.
- **Cool Web App**: A slick web app to show off our findings.
